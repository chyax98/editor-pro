# Editor Pro「插件融合」候选清单（低负担优先）

目标：从“必装/高口碑”插件中抽取**无需复杂配置**、且能显著提升编辑体验的能力，逐个评估并融入 Editor Pro（不复制他人代码，只复刻体验与交互）。

评估原则：
- 默认离线；不做“必须联网才能用”的核心功能
- 默认零学习成本；复杂能力只做成“可选增强”
- 优先键盘友好、无 UI/轻 UI、可直觉理解的功能

| 优先级 | 来源插件/方向 | 我们要的精华 | Editor Pro 方案（低负担） | 用户负担 | 状态 |
|---|---|---|---|---|---|
| T0 | Paste URL into selection | 选中文字粘贴 URL 变链接 | 默认开启；仅在有选区且剪贴板是 `http(s)` 才接管 | 低 | ✅ 已实现 |
| T0 | Typewriter Mode | 光标行固定在屏幕位置、聚焦写作 | 做成一个开关：`typewriter scroll` + 可选“高亮当前行” | 低/中 | ⬜️ |
| T0 | Outliner | 列表像 Workflowy 一样移动/折叠/缩进更顺滑 | 先做最小集：`Tab/Shift+Tab` 缩进、Alt+↑↓ 移动 list item | 低 | ⬜️ |
| T0 | Smart Typography | 引号/破折号/省略号等自动替换 | 只做“可逆 + 可关闭”的最小替换集（避免改写用户风格） | 中 | ⬜️ |
| T0 | Linter | 自动统一格式（尤其 YAML/空行/列表） | 不做全量规则；只做少量“安全规则”并默认关闭 | 中/高 | ⬜️ |
| T1 | Advanced Tables | 表格编辑像表格软件 | 继续补齐：列/对齐/格式化入口 + 更完善的 Tab 行为 | 中 | ⬜️（已部分） |
| T1 | Quiet Outline / Outline | 大纲导航更舒服 | 仅做轻量增强：当前标题高亮、快速折叠/展开、键盘跳转 | 低/中 | ⬜️ |
| T1 | Natural Language Dates | `tomorrow/next fri` 转日期 | 仅离线解析常见英文/中文短语，作为可选命令 | 中 | ⬜️ |
| T1 | Sentence Rhythm / Readability | 写作节奏/可读性提示 | 先不做实时渲染；做成一次性检查命令（可选） | 中 | ⬜️ |
| T2 | Style Settings | 可视化调样式 | 不做“全量主题配置”；只做少量 Editor Pro 自己的 CSS 变量开关 | 中 | ⬜️ |
| T2 | Tag Wrangler | 标签批量重命名 | 需要全库扫描，风险高；延后 | 高 | ⬜️ |

## 下一步建议（按“收益/成本/负担”排序）

1. Typewriter Scroll（T0）：写作幸福感极高，且可以做到“一个开关就完事”
2. Outliner 的最小集（T0）：只做缩进/移动/折叠的键盘体验，不做复杂视图
3. Advanced Tables 补齐入口（T1）：把已有算法“接到 UI/命令上”，让它真的可用
